경로 표시 방법
절대 경로 : 인터넷의 고유한 주소를 그대로 씀 다른 웹사이트의 페이지
.(점)표기법 : 리눅스나 윈도우 내에서 .은 현재의 위치를 의미함, ..은 상위폴더를 의미함
윈도우키 + R : 실행
cmd(command)입력, cd를 입력하면 현재의 위치를 알려줌. cd .. 을 입력하면 상위폴더를 보여줌.(경로처리할때 이러한 방식으로 처리함)
ex) ./img/image

테이블은 표 형태의 데이터를 표시하는데 사용된다. 테이블을 <table>태그로 정의된다.
HTML에서는 테이블의 하나의 행을 <tr> ... </tr>로 표현한다. tr은 "table row"의 약자이다. 하나의 데이터는 <td>... </td>요소로 표현한다.
표에서 학생이름, 중간고사, 기말고사와 같이 값이 어떤것인지 알려주는 부분 -> 테이블 헤더
웹에서 자료를 가져올때 테이블 헤더와 테이블 바디로 나눠서 생각.
colspan="5"로 하면 5열을 하나로 합칠 수 있음
rowspan="숫자"로 나눌 수 있음
<td> <th>로 나눠서 과목의 이름과 나머지를 구분해서 지정하니까 나중에 데이터 분석등을 할때 필요한 자료만 끌어와서 사용할 수 있음
