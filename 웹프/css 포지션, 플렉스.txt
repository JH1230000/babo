position을 이용해서 선택한 요소의 위치 설정 가능(배치)
- absolute는 스타일을 준 요소의 부모 요소를 기준으로 설정한 만큼 이동한다.
- relative는 스타일을 준 요소의 원래 위치를 기준으로 설정한 만큼 이동한다.
- fixed는 드래그 요소가 있을 때 전체 화면을 기준으로 설정한 만큼 이동한다.

요소 쌓임 순서
1. 요소에 position 값이 있는 경우 위에 쌓임 (기본값 static 제외)
2. 1번 조건이 같은 경우 z-index 숫자 값이 높을 수록 위에 쌓임
3. 1,2번 조건이 모두 같으면 html의 다음 구조일 수록 위에 쌓임

요소의 display
inline : 같은 줄에 옆으로 나열
inline-block : 인라인 블럭은 같은 줄에 옆으로 나열하는 인라인 + 크기 설정 가능한 블럭
block : 각각 다른 줄에 나열
flex : 플렉스 컨테이너에 포함된 요소의 정렬을 할 수 있다.
